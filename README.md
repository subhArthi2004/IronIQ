# 💪 IronIQ

IronIQ is a fitness web application built using **React**, **Vite**, and **Supabase**. It provides a secure authentication system with login, signup, and a protected dashboard. This project was created to learn modern web development and user authentication.

---

## 🚀 Features

- 🔐 User Signup
- 🔑 User Login
- 🛡️ Protected Dashboard
- 👤 User Authentication using Supabase
- ⚡ Fast and responsive interface built with React & Vite

---

## 🛠️ Technologies Used

- React
- Vite
- JavaScript (ES6+)
- React Router
- Supabase
- CSS

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/subhArthi2004/IronIQ.git
```

Go to the project folder:

```bash
cd IronIQ/client
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the **client** folder and add:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
